🌥️ 个人简介

  1. 一个树立终身学习，谦虚向他人学习态度的95后，努力变得更好。
  2. 瑾怀轩 出自《楚辞》中《九章·怀沙》的‘怀瑾握瑜兮’，避南公名讳，故取瑾怀，轩取义气度不凡，与众不同。以鞭策自己保持良好品性，向往美好，不断学习。记训于此‘明德，励志，博学，笃行’
  3. 博客目前使用的是 Hexo框架 和 Next主题 、 GitHub 、 coding 服务与国内免费的图床（放大图...可没说尺寸哦！） SM.MS 、 路过 和腾讯云的COS（当然宽裕的可以全部放在独角兽企业的服务器上）。域名使用的是个人姓氏拼音，好记。
  4. 本站只支持 Chrome、Firefox 等标准浏览器，不支持 IE 哦
  5. 懂一些前端知识，但主要是做 Java 后端开发 ，目前也在不断 看书 提升技术水平。但个人水平非常有限，平时在站内分享的博客难免内容有误，欢迎各位同行批评指正🤝，不胜感激！
